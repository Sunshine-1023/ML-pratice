# MLP 回歸模型實驗報告（房價預測）

## 一、實驗目的

- 建立 MLP 回歸模型，使用多個結構化特徵預測房價中位數 `MedHouseVal`。
- 在避免資料洩漏的前提下，完成訓練、驗證、測試流程。
- 使用 Early Stopping 提升訓練穩定性，並輸出可解讀圖表。

## 二、資料集與特徵

- 訓練檔案：`数据集/train_data.csv`
- 測試檔案：`数据集/test_data.csv`
- 目標變數：`MedHouseVal`
- 特徵（8 維）：
  - `MedInc`
  - `HouseAge`
  - `AveRooms`
  - `AveBedrms`
  - `Population`
  - `AveOccup`
  - `Latitude`
  - `Longitude`

## 三、資料前處理

### 1) 訓練/驗證/測試切分

- 原始訓練集再切分為訓練子集與驗證子集，比例 `8:2`。
- 使用：
  - `train_test_split(..., test_size=0.2, random_state=42)`
- 本次樣本數：
  - 訓練子集：`13209`
  - 驗證子集：`3303`
  - 測試集：`4128`

### 2) 標準化（避免量綱影響）

- 使用 `StandardScaler` 對輸入特徵與目標值分別標準化。
- 僅使用訓練子集 `fit`，驗證與測試只做 `transform`，避免測試資訊洩漏。
- 標準化公式：

\[
x'=\frac{x-\mu}{\sigma}
\]

其中 \(\mu\) 與 \(\sigma\) 為訓練子集統計量。

## 四、模型與訓練策略

### 1) 模型結構

- 使用多層感知器（MLP）：`8 -> 64 -> 32 -> 1`，中間激活函數為 `ReLU`，並使用 `Dropout(0.2)`。

### 2) 優化配置

- 損失函數：`SmoothL1Loss`
- 優化器：`AdamW(lr=1e-3, weight_decay=1e-4)`

### 3) Early Stopping（核心）

- `max_epochs = 5000`
- `patience = 80`
- `min_delta = 1e-5`
- 監控指標：驗證集 `val_loss (SmoothL1)`
- 策略：
  - 若 `val_loss` 創新低，保存 `best_state_dict`
  - 若連續 `patience` 輪無有效改善，提前停止
  - 訓練結束回載最佳權重

> 說明：整個提前停止判斷只用驗證集，不使用測試集。

## 七、結果分析

- 已完成標準化、資料隔離與 8:2 訓練/驗證切分，流程正確。
- Early Stopping 生效，可避免無效長訓練與過擬合風險。
- 驗證集表現穩定，但測試集 `R2` 偏低，顯示泛化能力仍有提升空間。
- 可能原因：
  - 線性模型表達能力有限，難以捕捉非線性關係
  - 資料中存在離群點，對 RMSE 影響明顯

## 八、結論與改進方向

本實驗已建立完整且規範的線性回歸流程，並可穩定輸出多種分析圖表。  
未來可從以下方向提升效果：

- 加入特徵工程（交互項、非線性變換）
- 採用正則化回歸（Ridge/Lasso）或樹模型（XGBoost/LightGBM）
- 對離群值做更穩健處理（如 clipping 或 robust 損失）
- 使用交叉驗證比較不同模型，再選擇最終方案
